what i've worked on

UROP multilingual computational analysis

a research project using multilingual robinson crusoe translations as a case study for computational analysis of historical texts, focusing on building a clean corpus from noisy scans and comparing patterns across languages.

S

our research group wanted to study multilingual robinson crusoe translations, but the texts only existed as uneven-quality scans scattered across digital libraries.

T

turn 1,200+ scanned pages into a clean, structured dataset that we could reliably use for nlp analysis, without spending forever on manual checking.

A

i sourced translations from hathitrust, google books, and public-library archives, then built a python + pandas ocr pipeline that converted the scans into structured dataframes. i also added automatic tests that flagged ocr/tokenization mismatches.

R

we ended up with a high-quality corpus ready for downstream nlp work and cut manual qa time by about 25%, making it much faster to experiment with new questions and models.

spotify bot

an automous program that handle on-demand, artist-specific track recommendations via slash commands.

S

friends on my Discord server wanted quick, personalized music recommendations, but bouncing between Discord and Spotify was clunky.

T

build a bot that could serve artist-specific Spotify track recommendations directly inside Discord on demand.

A

implemented an asynchronous Discord bot using discord.py integrated with the Spotify Web API and slash commands, and wrote a Selenium scraper to seed a JSON dataset with 1,000+ artist names.

R

server members can now get tailored track recommendations with a single command, and the automated scraper eliminates manual data entry when expanding the bot's catalog.

jain blocks

an interactive educational toy that pairs 3d-printed letter blocks with an fpga-based microprocessor, turning physical block placements into sounds and words to help kids practice early literacy skills.

S

early-grade literacy scores in the U.S. have declined, especially after remote learning, and young learners often struggle to stay engaged with traditional reading drills.

T

design an interactive toy that lets children physically build sight words with blocks, then hear letter names and words spoken back to them, giving teachers and parents a more engaging way to practice reading.

A

our team created JAIN blocks: 3D-printed letter blocks act like encoded switches in slots, wired through pull-down resistor circuits to an FPGA board

R

the final prototype successfully recognizes letter combinations, sounds out and speaks valid words, and gives immediate audio feedback when a spelling is wrong
